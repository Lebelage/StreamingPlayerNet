#Приложение позволяет скачивать .torrent файлы и одновременно проигрывать скачиваемое медиа(одиночные файлы .mkv, mp4 ... форматов).
#В правом верхнем углу располагается кнопка выбора .torrent файла(предне). После выбора начнется скачивание.
#Начать проигрывание медиа можно с помощью кнопки "Play". При запуске проигрывания кнопка меняет свой функционал на "Pause", что останавливает проигрывание на текущем моменте.
